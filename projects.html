Projects

Text Independent Speaker Recognition

Dr. Pradip Kumar Das , Associate Professor , Dept. of CSE , IIT Guwahati

Developing a text independent speaker recognition system by extracting features using MFCC and building the model for each speaker using Gaussian Mixture Model.

Support Vector Machines for Image Annotation

Dr. Arijit Sur , Associate Professor , Dept. of CSE , IIT Guwahati

Building a Support Vector Clustering model for content based image annotation, in which each cluster is represented by the radius of smallest sphere which encloses most of the CPAM feature vectors of the image in the mapped space.

Compiler for subset of C programs

Dr. Arnab Sarkar , Associate Professor , Dept. of CSE , IIT Guwahati

Developed a compiler in bison, which performs lexical analysis, syntactic analysis and finally generates intermediate code for subset of C programs.

Network Simulation using ns3

Dr. Sukumar Nandi , Associate Professor , Dept. of CSE , IIT Guwahati

Simulated TCP protocol for a given topology over both wired and wireless network using ns3.

Video Search Engine

Dr. Amit Awekar , Assistant Professor , Dept. of CSE , IIT Guwahati

Developed a video search engine similar to Youtube that works on the integration of SQL, MongoDB, Neo4j databases for efficient search across the database and giving the desired results to the end-user. Key Features include recommendation system, trending videos, blacklist videos, history.

Web Crawler

Dr. Santosh Biswas , Associate Professor , Dept. of CSE , IIT Guwahati

Developed a search engine for the intranet website of IITG which crawls the webpages and suggests the best possible results for the search terms. Key Features include Fuzzy Search, Advanced Search. Technologies Used : Django, Python.

File Sharing Portal

Dr. Santosh Biswas , Associate Professor , Dept. of CSE , IIT Guwahati

Developed a Dropbox based website which enables the user to share files, through link generated by website, by uploading them. Key Features include Link expiry, Integrity Check, Compression of files, Password protection for files, Login support, Cloud storage. Technologies Used : PHP, HTML, CSS, Javascript.

Student Information System

Dr. Sourabh Joshi, Department of CSE, IIT Guwahati

The project provides a prototype of a student information system , which makes it easy for the faculty to access relevant information about the student and to keep track of the student's academic performance and various other things. Technologies Used : Laravel (PHP frameworks), mySQL database.

Assembler-Linker-Loader

Dr. Santosh Biswas , Associate Professor , Dept. of CSE , IIT Guwahati

Developed assembler linker loader in java for a subset of C programs with detailed simulation of intermediate steps involved and reporting of syntax errors in the C program.
